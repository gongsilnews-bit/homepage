# 부동산 홈페이지 상세 구성 초안 (Draft)

이 문서는 모든 부동산 템플릿(원룸형, 상가/사무실형 등)이 공통적으로 가져가야 할 **'표준 홈페이지 화면 구성(섹션)'**과 각 섹션별로 **'어떤 내용을 바꿀 수 있게 만들 것인가(설정 항목)'**를 정리하기 위한 회의용 초안입니다.

---

## 🏗️ 1. 기본 화면 구성 (표준 섹션 레이아웃)

어떤 테마를 선택하든, 아래의 섹션 블록들을 레고 블록처럼 끄고 켜는(On/Off) 방식으로 조립할 수 있게 만듭니다.

### ① 상단 영역 (Header & GNB)
- **표시 내용:** 회사 로고, 메인 메뉴, 전체메뉴(햄버거 바), 로그인/회원가입, 퀵 문의전화
- **어드민 제어 항목:**
  - 로고 이미지 업로드
  - 퀵 문의전화번호 (예: 010-1234-5678)
  - 상단 메뉴 구성 여부 (어떤 메뉴를 보일지)

### ② 메인 배너 영역 (Hero Section)
- **표시 내용:** 사이트 접속 시 가장 먼저 보이는 큰 이미지/영상과 캐치프레이즈, 검색창
- **어드민 제어 항목:**
  - 배경 이미지(또는 영상) 업로드
  - 메인 카피(큰 글씨) & 서브 카피(작은 글씨) 텍스트 커스텀
  - 검색창 표출 여부 (On/Off)

### ③ 퀵 검색/테마 검색 영역 (Theme Search)
- **표시 내용:** 원룸, 투룸, 사무실, 무보증, 주차가능 등 아이콘 형태의 퀵 버튼
- **어드민 제어 항목:**
  - 부동산 주력 상품에 따른 버튼 노출 설정 (원룸 위주냐, 상가 위주냐)

### ④ 매물 큐레이션 영역 (Property Section)
- **표시 내용:** '추천 매물', '최신 매물', '급매물' 등 탭(Tab) 형태나 슬라이드 형태로 보여주는 매물 리스트
- **어드민 제어 항목:**
  - 노출할 카테고리 탭 (예: 추천, 최신, 전속)
  - 몇 개의 매물을 보여줄 것인지 (예: 4개, 8개)

### ⑤ 간편 상담/문의 영역 (Consultation)
- **표시 내용:** 이름과 전화번호만 남기면 콜백이 가는 폼 + 최근 상담 완료된 내역(신뢰도 상승용)
- **어드민 제어 항목:**
  - 섹션 사용 여부 (On/Off)
  - "성함과 연락처를 남겨주시면 즉시 연락드립니다" 와 같은 안내 문구 커스텀

### ⑥ 블로그/뉴스/공지사항 영역 (News Preview)
- **표시 내용:** 부동산 칼럼이나 뉴스, 공지사항의 최신 글 미리보기
- **어드민 제어 항목:**
  - 어느 게시판의 글을 메인에 끌어올지 설정

### ⑦ 오시는 길 & 파트너 (Location & Partners)
- **표시 내용:** 카카오맵 기반 부동산 찾아오는 길, 그리고 제휴 관공서(정부24, LH 등) 배너
- **어드민 제어 항목:**
  - 부동산 실제 주소 입력 (지도 핀 위치 자동 연동)
  - 주차 가능 여부 문구

### ⑧ 하단 영역 (Footer)
- **표시 내용:** 사업자 정보, 회사 소개, 대표전화, 각종 약관 링크, 카카오톡/블로그 링크
- **어드민 제어 항목:**
  - 사업자 등록번호, 대표자명, 주소, 전화번호
  - 블로그, 유튜브, 카카오채널, 인스타그램 주소 (URL)

## 💡 2. 대표님과의 회의 안건 (의사결정 필요)

위 구성에서 대표님의 실무 경험에 비추어 보았을 때, 방향성을 정해야 할 핵심 질문들입니다.

1. **필수 vs 선택:** 모든 중개사가 위 8개 섹션을 다 쓰지는 않을 것입니다. (예: "우리는 간편 상담 폼 안 쓸래"). **모든 섹션에 On/Off 스위치를 달아주는 방향**이 좋을까요?
2. **팝업이나 공지 띠 배너:** 이벤트(예: 수수료 반값 할인)를 진행할 때 화면 가운데 뜨는 **'팝업창'**이나, 최상단에 얇게 깔리는 **'띠 배너'** 기능도 홈페이지 구성에 기본으로 포함하는 게 좋을까요?
3. **영상 배경:** 메인 배너(Hero)에 사진 말고 **'유튜브 영상'이 배경으로 재생**되게 하는 기능은 트렌디해 보이지만 로딩이 느려질 수 있습니다. 도입하는 게 좋을까요?
4. **추가하고 싶은 독창적 섹션:** 위 8가지 외에 기존 프랜차이즈나 다른 부동산 홈페이지에서 **"이건 꼭 있어야 해!"** 라고 생각하셨던 특별한 코너가 있으신가요? (예: 직원 소개 코너 등)
